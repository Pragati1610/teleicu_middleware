# Teleicu Middleware
Middleware to help tunnel CCTV Streams and ONVIF APIs for TeleICU.

---

## Installation

In the project directory, you can run:

```
// Install dependencies
$ npm install

// Start server
$ npm start

// Start server - watch mode
$ npm run dev
```
